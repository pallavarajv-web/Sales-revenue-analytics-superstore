# Sales & Revenue Analytics-superstore Dashboard

## Project Overview
This project analyzes sales, profit, customer segments, regions, and product performance using the Superstore dataset.

## Tools Used
- Power BI
- Excel
- Data Analytics

## KPIs
- Total Sales
- Total Profit
- Total Orders

## Dashboard Features
- Sales by Region
- Sales by Category
- Profit by Category
- Profit by Segment
- Monthly Sales Trend
- Monthly Profit Trend
- Top 10 Products by Sales
- Sales by Sub-Category
- Interactive Filters and Slicers

## Key Insights
- Identified top-performing regions.
- Analyzed profitable product categories.
- Evaluated customer segment performance.
- Tracked monthly sales and profit trends.
